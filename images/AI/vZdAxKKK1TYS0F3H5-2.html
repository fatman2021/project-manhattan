Overview

Project Manhattan is a groundbreaking emulator that combines the power of the classic MOS 6510 CPU with modern GPU capabilities. Written in FreeBASIC, this system-on-chip (SoC) emulator brings new life to retro computing and opens up exciting possibilities for both gaming and business applications.

Key Features

  • Accurate MOS 6510 CPU emulation
  • Integrated GPU with GLSL support
  • Compatibility with C64 software
  • Extended instruction set for modern capabilities
  • Ring-0 and x86 low-level optimizations
  • Built-in assembler for direct machine code programming
  • Business application framework
  • Game development toolkit
